3. Les auto-globales de PHP

Cette séance étudie l’envoi et la récupération de données par formulaire, en méthode get et en méthode post, à travers une petite application. Celle-ci doit permettre de saisir un numéro via un formulaire, puis, pour les étudiants du groupe du numéro indiqué, de rentrer leurs notes. La moyenne des notes rentrées et la liste des notes non valides sont alors retournées. Deux scripts PHP de nom entrerNotes et notesEntrees seront à écrire.


1 Rappels sur les tableaux associatifs

Les tableaux associatifs permettent de stocker des ensembles de données en les indexant avec des clés non nécessairement numériques à la différence des tableaux classiques. Ils sont souvent utilisés en PHP et les exemples les plus connus sont certainement $_GET, $_POST, $_REQUEST qui permettent de récupérer la valeur associée à chaque entrée d’un formulaire selon la méthode HTTP avec laquelle il a été soumis. Cet exercice a pour but de se familiariser avec les tableaux associatifs et de saisir les différences avec les tableaux classiques.

  1. Créer une page fruits.php qui affiche dans une liste non numérotée et dans l’ordre alphabétique les éléments d’un tableau fruits contenant "banane", "pomme" et "litchie". On utilisera successivement la boucle while, la bouche for et la bouche foreach.
  2. Créer une page calories.php qui affiche dans un tableau de deux colonnes les éléments d’un tableau associatif calorie contenant les associations "banane"=>130, "pomme"=>300 et "litchie"=>30. On utilisera la boucle foreach et les éléments du tableau associatif seront affichés dans l’ordre croissant de leurs calories. Modifier la page pour que l’affichage respecte l’ordre alphabétique des noms de fruits.

Note : N’hésiter pas à implémenter et à utiliser les fonctions array_to_list et array_to_table vues lors du TD2.

Rappel : la fonction ksort() permet de trier un tableau associatif selon l’ordre croissant de ses valeurs d’index et la fonction asort() selon l’ordre croissant de ses valeurs.

2 Envoi controlé du numéro

Lors de la première exécution de entrerNotes (voir la séance 3 de Travaux Dirigés), un formulaire demande le numéro du groupe concerné. La soumission du formulaire se fait par la méthodes get. Ce formulaire est reproposé (avec un message d’erreur) si le numéro :

  • n’a pas été saisi,
  • ou est supérieur à une constante définie dans le script.
Solution Télécharger entrerNotes1.php

<?php
define('MAX_ID',12);

require_once('../2/entete.php');

function creerForm($val='') {
  return
    "<form action='' method='get'><fieldset>\n" .
    "<label>Numero de groupe : </label>\n" .
    "<input type='text' name='etu_id' value='$val' />\n" .
    "<input type='submit' />\n" .
    "</fieldset></form>\n";
}


echo entete("Saisir Notes");
echo "<body>\n";

if ( empty($_GET)  ) {
  echo creerForm();
} else {
  if ( isset($_GET['etu_id']) ) {
    $num = intval($_GET['etu_id']);
    if (!$num) {
      echo '<div>Le groupe ne peut etre vide</div>';
    } else if ( $num > MAX_ID  ) {
      echo '<div>Numero de groupe trop grand</div>';
    }
    echo creerForm($num);
  }
}

echo "</body></html>";

?>

5 Mise à jour d’un fichier

Nous supposons dans cet exercice que les données relatives aux étudiants sont stockées dans un fichier texte de plusieurs lignes ou chaque ligne contient le couple numéroEtudiant numéroTD (avec l’espace séparant le numéro étudiant et le numéro TD)

 Ecrire la fonction tabFromFich qui, étant donné un nom de fichier, renvoie le tableau associatif correspondant des couples (numéro de carte, numéro de TD)

 Ecrire la fonction TDdeEtudiant qui, étant donné un nom de fichier, et un numéro d’étudiant renvoie le TD où l’étudiant est inscrit et O s’il n’est pas inscrit.

 Ecrire la fonction ajoutEnFinFile qui, étant donné un nom de fichier, un numéro d’étudiant et un TD ajoute le couple (numéro de carte de l’étudiant, TD) en fin du fichier.

 Ecrire un script pour générer aléatoirement une trentaine de couples et tester ainsi le remplissage du fichier des inscrits. Contrôler le bon déroulement du script par affichage du fichier.

 Modifier le script inscrireEtudiant, pour qu’une fois le formulaire bien rempli et les données valides, et l’étudiant non encore inscrit, le couple (numéro de carte de l’étudiant, TD) soit rentré dans le fichier des inscrits.

<?php
/********************************************************************
 * Quelques précisions concernant cet exercice. On suppose que le fichier 
 * à lire est un fichier texte de plusieurs lignes ou chaque ligne contient
 * le couple numéroEtudiant numéroTD (avec l'espace séparant le numéro étudiant 
 * et le numéro TD) 
 *
 * */   

//Question 1
function tabFromFich($nomFich) {
    /*La fonction "file" fait que les éléments du tableau $lignes sont 
      des chaînes terminées par "\n".*/
   
  $lignes = is_readable($nomFich) ? file($nomFich) : array(); 

  // Il faut supprimer ces caractères parasites
   
  for ($i = 0; $i < count($lignes); $i++ ) {
       $lignes[$i] = rtrim($lignes[$i]);
   }
   
   /* Il faut décomposer chaque ligne en un tableau à deux éléments 
      (ici appelé $tabligne). On peut alors inscrire le couple clef-valeur 
      dans le tableau associatif  $clefVal créé pour l'occasion. 
   */
  $clefVal = array();
   foreach ( $lignes as $lg ) {
        $tabligne = explode(" ", $lg);
        $clefVal[$tabligne[0]] = $tabligne[1];
   }

   // et on renvoie comme valeur le tableau $Clef_Val  rempli.
   return $clefVal;
} // fin du texte de la fonction tabFromFich

//Question 2
function TDdeEtudiant($nomFichier, $numeroEtudiant){
  $tabClefVal = tabFromFich($nomFichier);
  if($tabClefVal[$numeroEtudiant])
    return $tabClefVal[$numeroEtudiant];
  else
    return 0;
}

//Question 3
function ajoutEnFinFile($nomFich,$nuEtu, $TD){
   $lignes = is_readable($nomFich) ? file($nomFich) : array(); 
   $lignes[count($lignes)]=$nuEtu." ".$TD."\n";
   $fichier =@fopen($nomFich,"w");
   foreach ($lignes as $lg){
     fputs($fichier, $lg);
   }
   fclose($fichier);
}//fin du texte de ajoutEnFinFile
